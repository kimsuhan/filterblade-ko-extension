# FilterBlade 한국어 번역 확장 프로그램

FilterBlade PoE2 페이지의 주요 UI 문구를 한국어로 바꾸는 크롬 확장 프로그램입니다.

## 설치 방법

1. 크롬 주소창에 `chrome://extensions`를 입력합니다.
2. 오른쪽 위의 `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램을 로드`를 누릅니다.
4. 이 폴더를 선택합니다: `C:\Users\k_yak\Documents\POE2-FilterBlade\filterblade-ko-extension`
5. `https://www.filterblade.xyz/?game=Poe2`를 새로고침합니다.

## 사용 방법

확장 프로그램 아이콘을 누르면 번역을 켜거나 끌 수 있습니다. 사이트가 화면을 동적으로 다시 그려도 새로 나타나는 문구를 계속 번역합니다.

## 번역 추가

새 문구를 추가하려면 `content.js`의 `exactTranslations` 또는 `phraseTranslations`에 영어 원문과 한국어 번역을 넣으면 됩니다.
